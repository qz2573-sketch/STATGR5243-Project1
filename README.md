# STATGR5243-Project1
# Project 1: Data Acquisition & Preprocessing for NYC Taxi Data

## Project Overview
[cite_start]This project focuses on the end-to-end data pipeline, including acquisition, cleaning, exploratory data analysis (EDA), preprocessing, and feature engineering[cite: 1, 6]. [cite_start]The objective is to analyze trip records from the NYC Taxi & Limousine Commission (TLC) to derive meaningful insights regarding urban transportation patterns[cite: 5].

## Data Source
The data is acquired from the official NYC TLC Trip Record Data page:
- **Source:** [NYC TLC Trip Record Data](https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page)
- [cite_start]**Method:** Web Scraping & API (using Python `requests` and `BeautifulSoup`) [cite: 15]
- **Scope:** July, August, September, and October 2025
- **Vehicle Types:** Yellow Taxi, Green Taxi, and For-Hire Vehicles (FHV)
